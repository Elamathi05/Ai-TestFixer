# AI TestFixer - AI Prompts Engineering Log

This file contains the details of prompts used during the design, implementation, and code generation of each component within the **AI TestFixer** self-healing test automation framework.

| # | Agent/Component | AI Tool | Prompt Summary | Purpose | Output |
|---|-----------------|---------|----------------|---------|--------|
| 1 | FailureDetector | Google Antigravity | Define an agent class to trap four target Selenium exceptions, log the failure metadata to SQLite using db_manager, and flag if self-healing should execute. | Exception analysis and filtering | Python class `FailureDetector` returning error dictionary and logging failure. |
| 2 | ScreenshotAgent | Google Antigravity | Implement a screenshot taking agent that saves the current state of a Selenium driver to a sanitized, Windows-safe filename in the screenshots directory. | Failure state capture | Python class `ScreenshotAgent` returning the file path of saved screenshot. |
| 3 | LogAgent | Google Antigravity | Create a log agent that writes structured logs to logs/{test_name}.log and updates test_history table with runtime status. | Local log logging and database insertion | Python class `LogAgent` writing to log files and returning log file paths. |
| 4 | DOMAnalyzer | Google Antigravity | Implement an agent that extracts page source and searches for element attributes (id, name, class) similar to a failed locator using difflib.SequenceMatcher. | Candidate elements analysis | Python class `DOMAnalyzer` returning list of sorted dictionary elements with score >= 0.4. |
| 5 | AIAgent | Google Antigravity | Implement an AI agent that constructs a JSON prompt with candidate elements, page title, and failed locator, querying a local Ollama REST API. | Generating replacement suggestions | Python class `AIAgent` connecting to Ollama, cleaning, and returning JSON. |
| 6 | SelfHealer | Google Antigravity | Create an agent to execute Selenium find_element retries using healed locator and update SQLite learned_fixes table on success. | Run-time healed execution | Python class `SelfHealer` returning execution status and saving fix mappings. |
| 7 | Validator | Google Antigravity | Design an agent to execute validation actions (click, type, visibility checks) on healed elements, calculate duration, and record via LogAgent. | Recovery verification | Python class `Validator` running element interaction and measuring response time. |
| 8 | LearningAgent | Google Antigravity | Create a learning database agent that checks if a fix exists, increments usage count if present, and inserts new records otherwise. | Repair logic memory persistence | Python class `LearningAgent` managing repair records. |
| 9 | PerformanceMonitor | Google Antigravity | Create a timing performance agent using time.perf_counter() that calculates rolling averages over 10 runs and flags anomalies. | Telemetry and anomaly check | Python class `PerformanceMonitor` monitoring execution speed. |
| 10 | RetryIntelligence | Google Antigravity | Build a retry runner implementing exponential backoff (2^attempt) that checks SQLite for learned fixes and falls back to AI suggestions. | Execution retry orchestrator | Python class `RetryIntelligence` coordinating retries and agent cycles. |
| 11 | AlertNotification | Google Antigravity | Implement an alert agent that queries failures in the last hour and sends SSL/STARTTLS email alerts on threshold breach. | Real-time system monitoring alerts | Python class `AlertNotification` dispatching emails and logging alerts. |
| 12 | AutoCleanup | Google Antigravity | Design a cleaning agent that runs every 10 runs to clean screenshots (>7 days), logs (>30 days), and database duplicates. | System storage maintenance | Python class `AutoCleanup` executing database queries and filesystem operations. |
| 13 | Demo Website | Google Antigravity | Create a premium styled dark-mode HTML login web page containing login button and comments simulating element modifications. | Target web application under test | Styled premium login portal page `website/login.html`. |
| 14 | Automated Test | Google Antigravity | Write login_test.py implementing headless Chrome Selenium, Trapping locator failures, triggering the self-healing workflow, and CLI override options. | Selenium execution framework | Complete automated script `tests/login_test.py` running tests and self-healing. |
| 15 | Streamlit Dashboard | Google Antigravity | Build a 9-page wide dark-mode Streamlit dashboard with autorefresh, ReportLab PDF exporter, Plotly graphs, and file filters. | Telemetry presentation UI | Streamlit app `dashboard/app.py` displaying live DB logs, screenshots, and metrics. |
